# `skills/` — 智能体技能说明（Markdown）

本目录存放 **供角色智能体阅读** 的说明文档（如技术组长工作流、各子角色行为约束）。运行时代码会按角色加载对应 Markdown，**不是** Python 模块。

**全部角色类型一览**（主角色 + 子角色标识符）见仓库根目录 [README.md → 多角色全景](../README.md#多角色全景)。

## 子目录

| 路径 | 说明 |
|------|------|
| `tech_lead.md` 等 | 根目录下主技能文件 |
| `roles/` | 子角色（如 reviewer、developer）的补充说明 |

## 轻度 vs 重度

- **轻度**：可只保留与你启用的角色相关的 `.md`；未配置的角色不会加载对应技能。  
- **重度**：通常保留完整技能树，并与飞书群内实际启用的 bot 角色一致，避免提示词与权限不匹配。

修改技能文本后 **无需** 重新安装依赖，重启服务进程即可生效（具体热更新行为以运行配置为准）。
